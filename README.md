<h1>Автоматизация сбора данных через Telegram Mini App</h1>

<h3>О проекте</h3>
Проект автоматизации процесса сбора и обработки данных через Telegram Mini App с последующей передачей информации в Google Sheets.  
<b>Цель проекта</b> — исключить ручной ввод данных и создать единый поток информации для дальнейшей аналитики.

#

Бизнес-задача

До внедрения:

- данные передавались вручную
- были ошибки ввода
- отсутствовал единый формат
- сложность контроля

После внедрения:

- единая форма
- автоматическая запись
- контроль статусов
- прозрачность процесса

#

Используемые инструменты

- Telegram Mini App
- Google Forms
- Google Sheets
- Google Apps Script
- Process automation
- Business analysis
